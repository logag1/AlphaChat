# Alpha Chat
- 데스크탑 채팅앱 알파챗

## Stack
- electron js
- react js

## Usage
- 현재 빌드 관련 유틸이 아직 완성되지 않았어요
- 소스 내의 api 서버 경로를 백엔드 서버 도메인으로 바꿔주세요
- 루트 디렉토리의 .ex.json 파일을 각각
`config.json` `msg.json` 으로 바꿔주세요
```
npm run server
npm run start
```
